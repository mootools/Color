Class: Color

Instances of the Color class serve as abstract reprentations of the color itself, and don't need to be transformed from one format to another. A single Color instance can have any component (red, green, blue, hue, saturation, lightness, brightness, alpha) updated regardless of the source, and all other components will be normalized automatically (e.g., setting the red component of a purplish color (7766CC) will update both the brightness and lightness components). If a Color is instanced using a hex value, it can have it's lightness component updated directly despite lightness being a HSL component. Further, the same color instance can output it's component values in any format without any extra conversions. When parsing a color representation, (either during instantiation, or via the parse method), the instance will attempt to determine the format supplied - if this is understood, toString calls will return the updated component values formatted in the same way (e.g., if instantiating a Color instance by passing an rgba string, subsequent calls to toString will also output an rgba string, but with updated values as transformations are applied to the color). The format of toString can be changed at any time, or direct formatting methods are available. Conversion methods (getRGB, getHex) are provided simply as sugar, and don't perform any actual transformations. They are not required for use or translation. The standard component parts are available as instance methods - if passed with an argument, the method will set the component value. Each component method returns the value as well (with or without setter argument). These components perform transformations and dispatch events, and can be used without any sugar to manage the Color instance.

Color Method: constructor

Syntax

var color = new Color([color]);

Arguments

  1. color - (mixed, optional) A string, number or object representation of a color.

Color

The constructor will use the parse method to try to convert a string, number or object to a valid color value, and can understand most human-readable representations as well as the return values of currentStyle or computedStyle (e.g., #FF0000, F00, rgb(100, 50, 25), rgba(100%, 50%, 0%, 0.8), hsla(360, 50%, 50%, 0.5), 16788595, 'orange', { red : 255, green : 100 })

Examples

var color = new Color('F90');
var color = new Color('#FF9900');
var color = new Color('orange');
var color = new Color('rgb(255, 103, 0)');
var color = new Color('rgba(100%, 50%, 0, 1)');
var color = new Color('hsl(360, 100%, 100%)');
var color = new Color(16788595);
var color = new Color({ red : 255, green : 103 });
var color = new Color(myElement.getStyle('color'));

Returns

  • (Color) A new Color instance.

Color Method: parse

Convert mixed variable to Color component properties, and adopt those properties.

Syntax

myColor.parse(value);

Arguments

  1. value - (mixed) A string, object or number color representation, including all valid CSS values. e.g., #FF9900, rgb(255, 153, 0), rgba(100%, 40%, 0%, 0.8); a hash with properties mapped to the Color instance e.g., red, green, saturation, brightness;another Color instance; a decimal color value; a named CSS color

Returns

  • (Color) This Color

Examples

var black = new Color();
black.parse('F90');
black.parse('#FF9900');
black.parse('black');
black.parse('rgb(255, 103, 0)');
black.parse('rgba(100%, 50%, 0, 1)');
black.parse('hsl(360, 100%, 100%)');
black.parse(16788595);
black.parse({ red : 255, green : 103 });
black.parse(myElement.getStyle('color'));

Color Method: hex

Set the hex component value of the color, updates all other components, and dispatches Event.UPDATED

Syntax

var somevar = myColor.hex([value]);

Arguments

  1. value - (string, optional) hex value to set (#000000 to #FFFFFF)

Returns

  • (string) The hex value of the Color instance.

Examples

var orange = new Color('#FF9900');
console.log(orange.getHex()); // outputs #Ff9900
orange.hex('#00FF00');
console.log(orange.getHex());  // outputs #00FF00

Color Method: format

Returns a tokenized string from the Color's component values

Syntax

myColor.format(string);

Arguments

  1. string - (string) The string to return, with tokens expressed as %token% that are replaced with component values. Tokens are as follows:
    • r : red
    • g : green
    • b : blue
    • h : hue
    • s : saturation
    • l : lightness
    • v : brightness
    • a : alpha
    • x : hex
    • d : decimal

Returns

  • (string) String argument with tokens replaced by this Color's mapped component values

Examples

var orange = new Color('#FF9900');
var message = 'This color has a red component value of %r% and a green component value of %g%';
console.log(orange.format(message));  // outputs This color has a red component value of 255 and a green component value of 153

Color Method: interpolate

Sets the invoking Color instance component values to a point between the original value and the destination Color instance component value, modified by the factor

Syntax

myColor.interpolate(destination, factor);

Arguments

  1. destination - (Color) A Color instance to blend with this Color
  2. factor - (Number) 0-1, where 0 is the origin Color and 1 is the destination Color, and 0.5 is halfway between. This method will "blend" the colors.

Returns

  • (Color) This Color

Examples

var orange = new Color('#FF9900');
var black = new Color(0);
orange.interpolate(black, 0.2);
console.log(orange.toHex());  // outputs #CC7A00

Function: Color.bind

Returns a Color instance bound to an object's property, set initially to the value of the property, and updating that property when the color is modified.

Syntax

var color = Color.bind(object, property);

Arguments

  1. object - (object) Object containing the property to update
  2. property - (string) Name of the property to update

Returns

  • (Color) A new Color instance, set to the value of the object's property.

Examples

var color = Color.bind(myElement.style, 'backgroundColor');
console.log(color.getHex());  // outputs the hex value of myElement.style.backgroundColor
color.hex('#FF0000');  // myElement.style.backgroundColor will display as blue
console.log(myElement.style.backgroundColor);  // outputs #FF0000
